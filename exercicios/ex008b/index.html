Outras format

Código fonte

Para escrever parecendo em código fonte, temos a taga <code>, sendo a tag <pre> usada para manter as quebras de linhas. Exemplo: 

            
    for i in list: 
        print(i)

Apertando shift e tab vc apaga vários espaços de uma vez só, tab sozinho adiciona um espaço
