Modelo de caixas

O que é

Todo elemnto visível em HTML é uma caixa

É possível aninhar os conteudos, uma caixa dentro de outra.

Tipos de caixa

Box-level

Uma nova caixa sempre se inicia em uma linha nova e ocupa a largura total da tela.
EX: DIV, H, P, MAIN, HEADER, NAV, ARTICLE, ASIDE, FOOTER, FORM, VIDEO.

Inline-level

Não pular pra próxima linha e não opupa o tamanho inteiro.
EX: SPAN, A, CODE, SMALL, STRONG, EM, LABEL, BUTTON, INPUT, SELECT.
